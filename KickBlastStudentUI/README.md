# KickBlastStudentUI

## Open project
1. Open Visual Studio 2022.
2. Click **Open a project or solution**.
3. Select `KickBlastStudentUI.sln`.

## Run
1. Set **KickBlastStudentUI** as startup project.
2. Press **F5** to run.
3. Login with default account below.
